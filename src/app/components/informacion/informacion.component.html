Información del proyecto

Hemos desarrollado una aplicación web para identificador diferentes tipos de flores a través de una red neuronal. En nuestro portal podrás indetificar una flor subiendo una foto de esta; o podrás identicarla en tiempo real con una cámara, incluso con la del móvil.

También es importante mencionar que se podrá identificar un número limitado de flores, que son las del dataset con la que se ha entrenado a la red neuronal. Estos tipos de flores estarán guardados en la base de datos junto con la información que se mostrará de cada tipo de flor. Hemos trabajado con la base de datos online que ofrece Google, llamada Firebase

Funcionalidades implementadas

  • Reconocimiento de flores a través de fotos
    • Mostrar información del tipo de flor
    • Link para comprar semillas de esa flor
  • Reconocimiento de flores en tiempo real
  • Ver lista de flores identificables
  • Añadir más flores a la base de datos (administradores)
